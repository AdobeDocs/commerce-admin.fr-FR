# Modèles de newsletter

Vous pouvez créer autant de modèles de newsletter que nécessaire à différentes fins. Vous pouvez envoyer une mise à jour hebdomadaire des produits, une newsletter mensuelle ou une newsletter annuelle dédiée aux vacances. Les modèles de newsletter peuvent être préparés avec le balisage HTML ou sous forme de texte brut. Contrairement à HTML, les newsletters en texte brut ne contiennent pas d’images, de texte enrichi ou de liens formatés. Dans la grille, la colonne Type de modèle indique si un modèle est HTML ou texte.

![Modèles de newsletter - Ajouter à la file d’attente des newsletters](./assets/newsletter-templates-grid.png){width="700" zoomable="yes"}

## Création d’un modèle de newsletter

1. Dans la barre latérale d’administration, accédez à **[!UICONTROL Marketing]** > _[!UICONTROL Communications]_>**[!UICONTROL Newsletter Template]**.

1. Pour ajouter un modèle, cliquez sur **[!UICONTROL Add New Template]**.

1. Renseignez les paramètres du modèle :

   - Par **[!UICONTROL Template Name]**, saisissez le nom pour la référence interne.

   - Par **[!UICONTROL Template Subject]**, décrivez l’objectif de la newsletter.

   - Par **[!UICONTROL Sender Name]**, saisissez le nom de la personne qui doit apparaître comme l’expéditeur de la newsletter.

   - Par **[!UICONTROL Sender Email]**, saisissez l’adresse e-mail de l’expéditeur de la newsletter.

   ![Informations sur le modèle de newsletter](./assets/newsletter-template-information2.png){width="600" zoomable="yes"}

   - Par **[!UICONTROL Template Content]**, cliquez sur **[!UICONTROL Show / Hide Editor]** pour afficher l’éditeur WYSIWYG et mettre à jour le contenu si nécessaire.

     Pour en savoir plus, voir [&#x200B; Utilisation de l’éditeur &#x200B;](../content-design/editor.md).

     >[!NOTE]
     >
     >Ne supprimez pas le lien de désabonnement au bas du contenu du modèle. Dans certaines juridictions, le lien est exigé par la loi.

   - Par **[!UICONTROL Template Styles]**, saisissez les déclarations CSS nécessaires pour formater le contenu.

1. Cliquez sur **[!UICONTROL Preview Template]** pour voir à quoi cela ressemble et apportez les modifications nécessaires.

1. Cliquez ensuite sur **[!UICONTROL Save Template]**.

   Après avoir enregistré un modèle, **[!UICONTROL Save As]** apparaît la prochaine fois que vous le modifiez. Il peut être utilisé pour enregistrer des variations du modèle sans remplacer l’original.

## Convertir le modèle en texte brut

1. En haut de la page, cliquez sur **[!UICONTROL Convert to Plain Text]**, puis sur **[!UICONTROL OK]** lorsque vous y êtes invité.

1. Pour prévisualiser la version en texte brut du modèle, cliquez sur **[!UICONTROL Preview Template]**.

   L’aperçu s’ouvre dans un nouvel onglet du navigateur.

1. Pour enregistrer la version en texte brut, cliquez sur **[!UICONTROL Save Template]**.

## Restaurer l’HTML

1. En haut de la page, cliquez sur **[!UICONTROL Return HTML Version]**.  

1. Pour prévisualiser la version HTML du modèle, cliquez sur **[!UICONTROL Preview Template]**.

   L’aperçu s’ouvre dans un nouvel onglet du navigateur.

1. Pour enregistrer la version d’HTML, cliquez sur **[!UICONTROL Save Template]**.

## Suppression d’un modèle de newsletter

1. Dans la barre latérale _Admin_, accédez à **[!UICONTROL Marketing]** > _[!UICONTROL Communications]_>**[!UICONTROL Newsletter Template]**.

1. Recherchez le modèle de newsletter à supprimer et ouvrez-le en mode d’édition.

1. Dans la barre de menus, cliquez sur le bouton **[!UICONTROL Delete Template]** .

1. Pour confirmer l’action, cliquez sur **[!UICONTROL OK]**.

## Colonnes de grille

| Colonne | Description |
|--- |--- |
| [!UICONTROL ID] | Identifiant numérique unique attribué à chaque modèle de newsletter |
| [!UICONTROL Template] | Nom de l’entité du modèle |
| [!UICONTROL Added] | Date de création de l’entité de modèle |
| [!UICONTROL Updated] | Date de la dernière mise à jour de l’entité de modèle |
| [!UICONTROL Subject] | Objet du modèle de newsletter |
| [!UICONTROL Sender] | Coordonnées de l’expéditeur |
| [!UICONTROL Template Type] | Type de modèle : `html` ou `text` |
| [!UICONTROL Actions] | **[!UICONTROL Preview]** : ouvre une fenêtre distincte pour prévisualiser le modèle <br>**[!UICONTROL Queue Newsletter]**: place le modèle de newsletter dans la file d’attente d’envoi. |
