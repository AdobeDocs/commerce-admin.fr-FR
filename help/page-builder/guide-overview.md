# [!DNL Page Builder] Guide d’utilisation

Ce guide est destiné aux administrateurs d’Adobe Commerce et de Magento Open Source. Il fournit des informations détaillées sur les [!DNL Page Builder] fonctionnalités, notamment une présentation en trois parties pour la création de composants de contenu de base. Cela suppose une compréhension de base du coeur [!DNL Commerce] configuration et .

[!DNL Page Builder] comporte deux zones destinées aux administrateurs de magasin :

- L’administrateur : utilisez cette zone pour accéder à l’interface utilisateur de configuration et utiliser les outils du créateur de pages.
- L’interface de ligne de commande : utilisez cet outil pour exécuter les tâches d’installation et de configuration du serveur principal.

## [!DNL Page Builder] informations sur les développeurs

[!DNL Page Builder] est installé avec Adobe Commerce 2.4.x et Magento Open Source 2.4.3 et versions ultérieures, avec toutes les fonctionnalités activées par défaut. Pour plus d’informations sur les modifications incluses dans les versions de module, voir [[!DNL Page Builder] Notes de mise à jour](release-notes.md). La variable [[!DNL Page Builder] Guide du développeur](https://developer.adobe.com/commerce/frontend-core/page-builder/) fournit des détails sur l’architecture et la personnalisation des modules.
