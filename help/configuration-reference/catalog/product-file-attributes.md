# [!UICONTROL Catalog] > [!UICONTROL Product File Attributes]

{{config}}

## [!UICONTROL Configure Allowed File Types and Size]

![Attributs du fichier de produit](./assets/file-validation.png)

| Champ | [Portée](../../getting-started/websites-stores-views.md#scope-settings) | Description |
|--- |--- |--- |
| [!UICONTROL Allowed File Extensions] | Global | Liste des types de fichiers autorisés, séparés par des virgules, tels que `pdf,doc,docx,txt`. Laissez vide pour utiliser les types de fichiers autorisés par défaut : pdf, doc, docx, xls, xlsx, ppt, pptx, txt, csv, zip. |
| [!UICONTROL Maximum File Size] | Global | Taille maximale autorisée du fichier en Mo, par exemple `20.0`. La limite par défaut est de 16 Mo. La taille de fichier maximale autorisée est de 31 Mo. |
