# Comparer les produits

Comparer les produits génère une comparaison détaillée et côte à côte de deux produits ou plus. Selon le thème, le lien Ajouter pour comparer peut être représenté par une icône ou un texte. Le bloc _Comparer les produits_ s’affiche généralement dans la barre latérale gauche ou droite d’une page de catalogue.

![Exemple de storefront - liste de comparaison](./assets/storefront-comparison-list.png){width="700" zoomable="yes"}

Contrairement au bloc [Produits récemment consultés/comparés](products-viewed-compared.md), l’administrateur n’inclut pas de paramètres de configuration supplémentaires pour la comparaison de produits.

## Comparer les produits sur le storefront

Il existe plusieurs façons d’utiliser la liste de comparaison sur le storefront.

### À partir des pages de catalogue

1. Le client trouve les produits qu’il souhaite comparer et clique sur le lien **[!UICONTROL Add to Compare]** pour chacun d’eux.

1. Permet d’accéder à une page de catégorie associée.

   Selon le thème et la mise en page, la barre latérale peut contenir un bloc _Comparer les produits_. Si tel est le cas, les éléments de la catégorie qui sont marqués pour comparaison sont répertoriés.

   Le client peut cliquer sur _Supprimer_ ( ![Icône Supprimer](../assets/icon-delete-x.png) ) pour n’importe quel produit afin de le supprimer du rapport de comparaison ou cliquer sur **[!UICONTROL Clear All]** pour supprimer tous les éléments et recommencer à zéro avec vos sélections de comparaison.

1. Effectue un clic sur **[!UICONTROL Compare]**.

1. Pour imprimer les informations de comparaison, cliquez sur **[!UICONTROL Print This Page]**.

1. Pour supprimer un seul produit de la page de comparaison, cliquez sur _Supprimer_ ( ![icône Supprimer](../assets/icon-delete-x.png) ).

### À partir d’un message de notification

1. Lorsqu’un client ajoute un produit à une liste de comparaison, la page affiche un message de notification.

1. Dans la notification affichée en haut du message, cliquez sur le lien _liste de comparaison_.

   ![Notification de comparaison de produits](./assets/notification-comparison-list.png){width="700" zoomable="yes"}

Cette action redirige le client vers la liste de comparaison dans laquelle il peut accéder à des actions supplémentaires.

### À partir du bloc _Comparer des produits_

1. Le client trouve les produits qu’il souhaite comparer et clique sur le lien **[!UICONTROL Add to Compare]** pour chacun d’eux.

1. Dans l’en-tête près du champ de recherche, cliquez sur le lien _Comparer des produits_.

   ![En-tête Comparer des produits](./assets/compare-products-header.png){width="700" zoomable="yes"}

### Depuis le tableau de bord Mon compte

1. Le client ajoute les produits nécessaires à la liste de comparaison.

1. Accède à **[!UICONTROL My Account]**.

1. Dans le bloc _Comparer les produits_, clique sur **[!UICONTROL Compare]**.

   ![Bloc Comparer des produits dans le tableau de bord du compte client](./assets/my-account-compare-block.png){width="700" zoomable="yes"}

## Actions supplémentaires de la liste de comparaison

| [!UICONTROL Action] | Description |
|------|-----------|
| ![Icône Supprimer](../assets/icon-delete-x.png) | Supprime un seul élément de la liste de comparaison. |
| **[!UICONTROL Add to Cart]** | Ajoute un produit au panier. Si le produit possède des configurations, la page redirige le client vers la page produit où il sélectionne les options configurables, puis clique sur **[!UICONTROL Add to Cart]**. |
| _Icône Liste de souhaits_ | Ajoute un produit à une liste de souhaits (la fonctionnalité de liste de souhaits doit être activée dans la configuration du magasin). |
| _Imprimer Cette Page_ | Imprime la page de liste de comparaison. |
